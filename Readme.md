- This extension allows you to generate random string and UUID.
- You can choose multiple of following options when create new string:
    1. Lowercase Letters
    2. Uppercase Letters
    3. Numbers
    4. Speacial Characters

## Installation

- Go Chrome Web Market with [this link](https://chrome.google.com/webstore/detail/random-stringpassword-gen/jgoihhefghijiapfdoakebmfjjbbhoph) and use "Add to Chrome" button to install.
